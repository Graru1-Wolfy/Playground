# tf2sim

Python tick simulation of TF2 Soldier rocket jumping, ported from [TF2Simulator](https://github.com/bjorn-martinsson/TF2Simulator) (MIT).

## Install

```bash
pip install -e "packages/tf2sim[dev]"
```

## Run tests

```bash
cd packages/tf2sim && pytest
```

## Optional visualizer

```bash
pip install -e "packages/tf2sim[viz]"
```
