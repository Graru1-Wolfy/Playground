# Raw Ore Grinder

A minimal NeoForge 1.21.1 mod that adds a simple raw ore grinder.

## Features

- **Raw Ore Grinder** block — right-click with raw iron, copper, or gold to produce **2–4 Raw Dust**
- **Raw Dust** — smeltable and blastable into iron nuggets
- Advancement: **grind! Ya' Awesome!** (unlocked on first grind)

## Build

```bash
./gradlew build
```

Output JAR: `build/libs/raworegrinder-1.0.0.jar`

## Usage

1. Craft a Raw Ore Grinder (iron, cobblestone, grindstone).
2. Right-click the grinder while holding raw ore.
3. Smelt or blast Raw Dust in a furnace or blast furnace.
